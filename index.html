Noise & Level Units

dB = 10·log(P₂/P₁) = 20·log(V₂/V₁)
dBm = 10·log(P / 1mW)
dBV = 20·log(V / 1V)
  • Noise Figure (NF) = 10·log(F); F = SNR_in/SNR_out; ideal F = 1 (0 dB)
  • Thermal noise: P_n = kTB (k=1.38×10⁻²³ J/K); −174 dBm/Hz at 290 K
  • Cascade NF: F_total = F₁ + (F₂−1)/G₁ + (F₃−1)/(G₁G₂) + …
  • Noise types: thermal, shot, flicker (1/f), burst, avalanche, intermodulation
  • Image frequency: f_image = f_s ± 2·f_IF (superheterodyne)
Pad input = output level + pad loss (dB)

Amplitude Modulation (AM)

m = (V_max−V_min)/(V_max+V_min)
P_T = P_c(1 + m²/2)
BW = 2·f_m
  • Each sideband power = m²·P_c/4 (at m=1 → 25% of P_c each)
  • Over-modulation (m>1): splatter, distortion
  • SSB (J3E): saves 2/3 power, BW = f_m; carrier suppressed by balanced modulator
  • DSB-SC (H3E): carrier suppressed; double sideband, no carrier
  • VSB: used in TV (vestigial sideband)
USB = f_c + f_m   LSB = f_c − f_m
Power saving (carrier suppressed, m=1) = 66.7%

Frequency Modulation (FM)

m_f = Δf / f_m
BW ≈ 2(Δf + f_m) [Carson]
DR = Δf_max / f_m(max)
  • Commercial FM: Δf_max = ±75 kHz; BW = 200 kHz; 88–108 MHz band
  • Pre-emphasis 75 μs: boost high-freq audio before TX; de-emphasis at RX
  • Capture effect: stronger FM signal suppresses weaker one on same freq
  • FM stereo pilot tone = 19 kHz; L−R subcarrier = 38 kHz DSB-SC
  • SCA = 67 kHz subcarrier (background music services)
  • Emission designation: F3E = FM telephony

Digital Communications Fundamentals

f_s ≥ 2·f_max (Nyquist)
C = B·log₂(M) bps
C_max = B·log₂(1+S/N) [Shannon]
  • PCM bits/sample: n = log₂(L) where L = quantization levels
  • Quantizing noise SNR: ≈ 6.02n + 1.76 dB
  • Delta modulation: 1 bit/sample; slope overload when signal changes too fast
  • BPSK: 1 b/symbol; QPSK: 2 b/symbol; 8-PSK: 3; 16-QAM: 4; 64-QAM: 6
  • Baud = symbols/sec; bps = baud × bits/symbol
  • ASCII: 7 bits (128 chars); EBCDIC: 8 bits; Baudot: 5 bits
  • Hamming distance: min # of bit errors to convert one codeword to another
